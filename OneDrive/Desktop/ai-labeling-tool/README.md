# 🧠 ProLabel AI — Data Annotation Tool

> Building the data layer behind AI systems.

A simple and interactive **Streamlit-based data labeling tool** that allows users to upload CSV files, annotate rows, and export labeled datasets for machine learning workflows.

---

## 🚀 Why This Matters

Most AI systems focus on training models.

But **high-quality labeled data** is the real bottleneck.

This tool provides a clean interface to:
- Inspect data row by row
- Assign labels efficiently
- Track labeling progress

---

## ⚙️ Features

- 📂 Upload CSV datasets  
- 🏷️ Custom label creation (dynamic categories)  
- ⚡ Fast annotation with radio selection  
- 📊 Real-time progress tracking  
- 🔁 Navigate between rows (Previous / Next)  
- 💾 Export labeled dataset instantly  

---

## 🖥️ Built With

- **Python**
- **Streamlit**
